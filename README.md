# Платформа для работы с банком задач по кибербезопасности
Это моя дипломая работа Реализация прототипа платформы для работы с банком задач по кибербезопасности
Десктопное приложение для управления, фильтрации, перевода и локального запуска задач формата Capture The Flag (CTF). Программа выступает в роли умного лаунчера и системы управления знаниями для ИБ-специалистов. 
В папаке Demo-4 находяться файлы программы, main.py отвечает за запуск прототипа. В папке data находяться csv файлы для чтения прототипом. В Demo-4/data находяться файлы коллекций и файлы CTF-турниров. В папке backups храняться резевные копии измененных файлов. Download.py код для скачивания турниров и создания csv файла для него.

## 🛠 Требования
* Python 3.9+ 
* pip (пакетный менеджер)
* ОС: Windows (рекомендуется), Linux или macOS

## 🚀 Как запустить

1. Клонируйте репозиторий:
```bash
git clone [https://github.com/BelutijNazar/CTF-book.git](https://github.com/BelutijNazar/CTF-book.git)
cd CTF-book
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите проект:
```bash
python main.py
```
